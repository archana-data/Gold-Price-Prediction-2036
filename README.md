# Gold Price Prediction in India for 2036

This project predicts the gold price in India for the year 2036 using Machine Learning models.

## Dataset
- Source: Kaggle (Gold Price Dataset 1978–2023)
- Currency used: INR

## Machine Learning Models Used
- Linear Regression
- Polynomial Regression

## Tools & Technologies
- Python
- Pandas, NumPy
- Matplotlib
- Scikit-learn

## Output
- Predicted Gold Price (2036): ₹1,34,620 (Linear Regression)
- Predicted Gold Price (2036): ₹3,40,084 (Polynomial Regression)

---

## Conclusion
The project demonstrates that gold prices in India follow a non-linear increasing trend. Among the models used, **Polynomial Regression** provides better accuracy for long-term prediction and is chosen as the final model for forecasting the gold price in 2036.
